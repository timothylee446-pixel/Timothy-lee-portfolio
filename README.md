## Setup

1. Create a new GitHub repo (e.g. `timothy-lee-portfolio`)
2. Copy all files into the repo
3. Go to **Settings → Pages → Source** and select `main` branch
4. Your site will be live at `https://<username>.github.io/timothy-lee-portfolio/`

## Notes

- Image URLs currently point to Google's servers. For reliability, download them and place them in `assets/images/`, then update the paths.
- Videos are linked to YouTube / Google Drive — those links will continue to work as-is.
